# 🌸 鲜花销售系统

基于 SpringBoot + Vue3 的在线鲜花销售平台

## 技术栈

- **后端**: SpringBoot, MyBatis, MySQL, Redis
- **前端**: Vue3, Vite, Element Plus
- **部署**: Docker

## 功能特点

- 用户登录/注册
- 鲜花浏览与搜索
- 购物车与订单
- 管理后台

## 快速启动

```bash
# 克隆项目
git clone https://github.com/Guoyu-Coder/flower-system.git

# 后端
cd backend
mvn spring-boot:run

# 前端
cd frontend
npm install
npm run dev

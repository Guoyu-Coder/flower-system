## 🎯 快速启动步骤

### 步骤 1：启动 Redis（如未运行）

```bash
# Windows
redis-server.exe
```

### 步骤 2：启动后端

```bash
cd flower-shop-backend
mvn spring-boot:run
```

等待看到：`Started FlowerShopApplication in X seconds`

### 步骤 3：启动前端

```bash
cd flower-shop-frontend
npm install  # 首次运行需要
npm run dev
```

### 步骤 4：测试功能

1. 打开浏览器访问商城
2. 右下角点击「🤖 AI导购」按钮
3. 输入测试问题

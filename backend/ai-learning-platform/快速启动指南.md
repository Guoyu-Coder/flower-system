# 智学星途-AI智能学习平台 快速启动指南

## 一、项目简介

智学星途是一个基于DeepSeek大模型的企业级AI Agent智能学习平台，具备独立思考、任务规划、用户感知、长期记忆、自主决策能力的学习专属AI Agent。

## 二、技术架构

### 后端技术栈
- **核心框架**: SpringBoot 3.2.5 + Java 22
- **持久层**: MyBatis-Plus 3.5.7
- **数据库**: MySQL 8.0
- **缓存**: Redis 7.x
- **认证**: JWT
- **AI**: DeepSeek API + 自研Agent

### 前端技术栈
- **框架**: Vue3 + Composition API
- **构建**: Vite 5
- **状态**: Pinia
- **UI**: Element Plus
- **图表**: ECharts 5

## 四、快速启动

### 4.1 数据库配置

1. **创建数据库**
```bash
mysql -u root -p
CREATE DATABASE ai_learning_platform DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
EXIT;
```

2. **导入数据**
```bash
cd ai-learning-platform/sql
mysql -u root -p ai_learning_platform < init.sql
```

### 4.2 修改配置文件

编辑 `ai-learning-system/src/main/resources/application.yml`:

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/ai_learning_platform?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: your_mysql_username
    password: your_mysql_password
  
  data:
    redis:
      host: localhost
      port: 6379
      password: your_redis_password
```

### 4.3 配置DeepSeek API

设置环境变量或修改配置文件:
```bash
# Linux/Mac
export DEEPSEEK_API_KEY=your_api_key

# Windows PowerShell
$env:DEEPSEEK_API_KEY="your_api_key"
```

或在 `application.yml` 中配置:
```yaml
deepseek:
  api:
    key: your_api_key
```

### 4.4 启动后端服务

```bash
cd ai-learning-platform
mvn clean install
cd ai-learning-system
mvn spring-boot:run
```

或使用IDE直接运行 `AiLearningApplication.java`

### 4.5 验证后端启动

访问以下地址验证:
- API服务: http://localhost:8080
- Swagger文档: http://localhost:8080/swagger-ui.html

## 五、前端启动（可选）

### 5.1 创建前端项目

```bash
npm create vite@latest vue-learning-platform -- --template vue
cd vue-learning-platform
npm install
```

### 5.2 安装依赖

```bash
npm install vue-router@4 pinia element-plus axios echarts
```

### 5.3 配置代理

在 `vite.config.js` 中配置代理:

```javascript
export default defineConfig({
  plugins: [vue()],
  server: {
    port: 3000,
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true
      }
    }
  }
})
```

### 5.4 启动前端

```bash
npm run dev
```

访问: http://localhost:3000

## 八、API接口说明

### 用户接口
- `POST /api/user/register` - 用户注册
- `POST /api/user/login` - 用户登录
- `GET /api/user/info` - 获取用户信息
- `PUT /api/user/info` - 修改用户信息
- `PUT /api/user/password` - 修改密码

### AI接口
- `POST /api/ai/generate/questions` - AI智能出题
- `POST /api/ai/grade` - AI智能批改
- `POST /api/ai/diagnose` - 错题AI诊断
- `POST /api/ai/chat` - AI答疑对话
- `POST /api/ai/analyze/learning` - 学情AI分析

### 题库接口
- `GET /api/question/list` - 题目列表
- `POST /api/question` - 创建题目
- `PUT /api/question` - 更新题目
- `DELETE /api/question/{id}` - 删除题目

## 九、常见问题

### Q1: 数据库连接失败
**解决**: 检查MySQL服务是否启动，确认用户名密码正确

### Q2: Redis连接失败
**解决**: 检查Redis服务是否启动，确认host和port配置正确

### Q3: AI功能无法使用
**解决**: 
1. 确认DeepSeek API Key已正确配置
2. 检查网络连接是否正常
3. 查看日志中的错误信息

### Q4: 端口被占用
**解决**: 修改 `application.yml` 中的 `server.port` 为其他端口

## 十、部署说明

### 生产环境部署

1. **构建项目**
```bash
mvn clean package -DskipTests
```

2. **部署JAR包**
```bash
java -jar ai-learning-system/target/ai-learning-system-1.0.0.jar
```

3. **配置环境变量**
```bash
export SPRING_PROFILES_ACTIVE=prod
```

### Docker部署（可选）

```dockerfile
FROM openjdk:22-slim
COPY target/ai-learning-system-1.0.0.jar app.jar
ENTRYPOINT ["java","-jar","app.jar"]
```
